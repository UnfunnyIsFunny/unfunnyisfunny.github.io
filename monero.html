Mining Monero for Monero-Chan

			


			I mined Monero for a couple of days as an experiment, where I was not really looking for profit, but rather for experience with the software used. 
			I used the XMRig miner and the P2Pool tool (The mini version, since I do not have a high hashrate) to mine in a peer-to-peer fashion. 
			First I downloaded Monero CLI to setup my wallet. Then I ran the following commands (inside the respective program's directory) to setup my miner:


First off, to sync monerod:

/monerod --zmq-pub tcp://127.0.0.1:18083 --out-peers 64 --in-peers 32 --add-priority-node=node.supportxmr.com:18080 --add-priority-node=nodes.hashvault.pro:18080 --disable-dns-checkpoints --enable-dns-blocklist


Then to sync P2Pool:

./p2pool --host 127.0.0.1 --wallet (Your wallet name) --mini


Finally to run XMRig CPU Miner:

sudo ./xmrig -o 127.0.0.1:3333


I created a small script to run these commands automatically, but I don't plan on mining again any time soon.
