Setting up a Windows 11 VM using KVM and Virt-Manager

This is a quick guide on how to setup a Windows 11 VM with near native performance using KVM + Virt-Manager and its hardware passthrough features. Here are the specifications of the PC I used to run this VM on:
  • Motherboard: ROG STRIX X570-E GAMING WIFI II
  • CPU: AMD Ryzen 9 5900X (24) @ 4.200GH (boosted using the ASUS AI Tweaker, not overclocked)
  • Host GPU: NVIDIA GeForce GTX 1080 Ti (using proprietary NVIDIA drivers)
  • Guest GPU (Passthrough): NVIDIA GeForce GTX 1080 (using proprietary NVIDIA drivers)
  • RAM: Corsair CMW32GX4M4K4000C19 Vengeance RGB PRO 32GB
  • OS: Artix Linux x86_64
This guide is based off of the video made by Kenny from the channel Mental Outlaw.
